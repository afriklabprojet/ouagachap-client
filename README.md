# OUAGA CHAP - Application Client

Application mobile Flutter pour les clients de OUAGA CHAP - Service de livraison rapide à Ouagadougou, Burkina Faso.

## Installation

```bash
flutter pub get
flutter run
```

## Fonctionnalités

- Création de compte par téléphone + OTP
- Commander une livraison
- Suivi en temps réel
- Historique des commandes
- Paiement Mobile Money (Orange Money, Moov Money)
- Notation des coursiers
